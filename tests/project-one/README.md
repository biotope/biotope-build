## test project for biotope-build v8

Quick start (linux/osx):
- npm i
- cd node_modules/\@biotope/
- rm -rf build
- ln -s ../../../../ ./build
- cd ../../
- Happy testing

Where to start testing:
- localhost:8000
  - includes 2 components (react and biotope-element)
- localhost:8000/resources
  - open console
